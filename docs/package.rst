Documentation
=============

The documentation page lists out all of the relevant classes 
and functions in a package.

Package
-------

The package section documents all of the code in the ``__init__.py``
file. If you put any code in that file, it will show up
here. 

.. automodule:: package
   :members:

Module
^^^^^^

The module section documents all of the code in a particular 
Python file. Specifically, all of the code below is
documented from the ``module.py`` file.

   :members:
